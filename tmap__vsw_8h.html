Performs alignment in the sequencing direction. This will update query_end and target_end in the results.

Parameters:
vswthe query in its vectorized form
querythe query sequence
qlenthe query sequence length
targetthe target sequence
tlenthe target sequence length
resultthe structure in which to store the results
overflowreturns 1 if overflow occurs, 0 otherwise
score_thrthe minimum scoring threshold (inclusive)
directionhow to break ties
Returns:
the alignment score

direction explains how to break ties. If direction = 0, then query_end needs to be as small as possible. If direction = 1, then query_end needs to be as large as possible. In both cases, if there are still several possibilities, choose the one with the largest value of target_end among them).
