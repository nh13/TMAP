Detailed Description

API for BWT Index Lookups

This API facilitates a secondary hash into the BWT

hi is set to TMAP_BWT_INT_MAX if the offset is greater than the hash width. If l is set to TMAP_BWT_INT_MAX then that value is unavailable.
